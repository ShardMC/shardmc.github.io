# α0.1 is here!

Let's goooooo! This alpha has basically everything what 1.0 will have (unless we change the architecture, but that's very unlikely).
So basic components such as [_Amethyst_](https://github.com/shardmc/amethyst) and [_Echo_](https://github.com/shardmc/echo) are not going to change their APIs (except for adding new features).

# Alpha 0.1? Doesn't sound promising.

Well, it _should_ sound promising.
Why? Well, α0.1 is already playable (check our [Discord](https://discord.gg/CqVAzakrqx) to get a download of _Amethyst_, and launch it as a normal minecraft server).

# What's next?

Next, is β0.1 (wow). There is some work to be done in both _Amethyst_ and _Echo_... But where did the core part go you might ask, well, that is also planned for β0.1. Check our roadmap or issues tab in our repositories to get more information on current bugs and features that are going to be added.
